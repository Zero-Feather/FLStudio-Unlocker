## 教程

> **须知：**该补丁仅适用于安装包名为"flstudio_win64_20.9.2.2963.exe"的FL Studio版本

虽然我在压缩文件中提供了一份原作者的英文教程，但我决定还是在这里提供一份汉化版本：

1. 下载Unlocker.zip文件，并解压
2. 双击FLRegKey.Reg将其添加至注册表
3. 复制Patch中的文件到FL Studio的根目录并覆盖
4. 享受它吧
